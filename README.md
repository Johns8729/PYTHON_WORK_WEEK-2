# 🛒 Inventory System (Python)

## 📌 Project Title

Inventory Management System with Controlled Loop

---

## 📖 Description

This project is a Python-based inventory management system that allows users to register products, track quantities, and calculate basic statistics.

Unlike traditional implementations, this system **does not use `while True`**, but instead uses a controlled loop (`while continuar == "si"`) to manage program execution based on user input.

The program allows the user to:

* Add products to the inventory
* Display all stored products
* Calculate total inventory value and quantity
* End the program with a final daily summary

All products are stored in a **list of dictionaries**, making the structure simple and efficient.

---

## ⚙️ Features

* ✅ Input validation (name, price, quantity)
* ✅ Controlled loop using `"si"` / `"no"`
* ✅ Data storage using lists and dictionaries
* ✅ Modular functions
* ✅ Error handling with `try/except`
* ✅ Final day summary
